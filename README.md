**This is a sample node app to use websocket for chat**

**Setup app**
1. Clone the repo
2. Install node ( v20.17.0 )
3. Run `npm install`
4. Run `node server.js`
5. Go to `http://localhost:3003/`
6. Hit `Send Message`

**Note**
:information_desk_person:
Check `public/index.html` to see the integration
